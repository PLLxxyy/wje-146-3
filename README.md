# 社区宠物互助平台

一个面向社区的宠物互助平台，支持宠物档案管理、宠物广场、寄养互助、即时聊天、寻宠启事等功能。

## 技术栈

- **前端**: Vite + React 18 + TypeScript
- **后端**: Express + TypeScript + better-sqlite3
- **认证**: JWT + bcryptjs
- **开发工具**: concurrently 同时启动前后端

## 快速启动

```bash
npm run install:all
npm run dev
```

前端运行在 http://localhost:5173，后端运行在 http://localhost:3000。前端通过 Vite 代理 `/api` 到后端。

## 功能说明

- **注册登录**: 用户名密码注册，JWT 令牌认证
- **宠物档案**: 填写宠物名字、品种、年龄、照片、疫苗情况和注意事项
- **宠物广场**: 展示所有宠物，支持按猫/狗/异宠分类筛选，按品种名搜索
- **寄养需求**: 发布寄养需求（时间段、宠物信息、特殊要求），申请者填写养宠经验和家庭环境
- **寄养申请**: 发布者查看申请列表并选择寄养人，确认后双方进入聊天
- **聊天功能**: 寄养双方在确认后实时沟通寄养细节
- **寻宠启事**: 发布丢失宠物信息（照片、走失地点时间、联系方式），首页轮播展示，找到后标记已找回
- **个人中心**: "我的宠物"、"我的寄养"、"寻宠记录"三个 Tab 管理
- **寄养评价**: 寄养结束后发布者可给寄养人打分写评价

## 注意事项

- 数据库文件存储在 `server/data/pet.db`，首次启动自动建表和种子数据
- 照片上传转为 base64 存储在数据库中，生产环境建议使用 OSS
- 种子数据包含 user1 和 user2 两个测试账号及示例宠物数据
- 样式全部写在 `index.html` 的 `<style>` 标签中，无额外 CSS 文件
